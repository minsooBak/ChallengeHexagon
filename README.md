#### 🤝 팀 규칙 
1. 스크럼 오전 10시 / 오후 7시 30분
	 오전 : 작업할 내용 짧게 나누고 점검
	 오후 : 작업한 내용 나누고 보완할 점 의논

2. 말 예쁘게 하기

3. 퇴실 잊지 말기

4. 🕘 열공 중 인증하기 잊지 말기

5. 점심시간 1시~2시, 저녁시간 6시~7시

6. 물이나 화장실은 채팅만 남기고 갔다오기

7. 집중 안되거나 졸리면 10분정도 리프레쉬 하고 오기
